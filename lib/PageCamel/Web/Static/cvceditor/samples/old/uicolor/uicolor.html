This sample shows how to use the UI Color picker toolbar button to preview the skin color of the editor. Note:The UI skin color feature depends on the CVCEditor skin compatibility. The Moono and Kama skins are examples of skins that work with it.

If the uicolor plugin along with the dedicated UIColor toolbar button is added to CVCEditor, the user will also be able to pick the color of the UI from the color palette available in the UI Color Picker dialog window.

To insert a CVCEditor instance with the uicolor plugin enabled, use the following JavaScript call: 

CVCEDITOR.replace( 'textarea_id', {
    extraPlugins: 'uicolor',
    toolbar: [ [ 'Bold', 'Italic' ], [ 'UIColor' ] ]
});
