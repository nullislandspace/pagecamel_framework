CVCEditor Samples » Append To Page Element Using JavaScript Code

This sample is not maintained anymore. Check out the brand new samples in CVCEditor SDK.

The CVCEDITOR.appendTo() method serves to to place editors inside existing DOM elements. Unlike CVCEDITOR.replace(), a target container to be replaced is no longer necessary. A new editor instance is inserted directly wherever it is desired. 

CVCEDITOR.appendTo( 'container_id',
    { /* Configuration options to be used. */ }
    'Editor content to be used.'
);
