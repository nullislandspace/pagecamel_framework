CVCEditor Samples » Create and Destroy Editor Instances for Ajax Applications

This sample is not maintained anymore. Check out its brand new version in CVCEditor SDK.

This sample shows how to create and destroy CVCEditor instances on the fly. After the removal of CVCEditor the content created inside the editing area will be displayed in a <div> element.

For details of how to create this setup check the source code of this sample page for JavaScript code responsible for the creation and destruction of a CVCEditor instance.

Click the buttons to create and remove a CVCEditor instance.
