CVCEditor Samples » Using the Placeholder Plugin

This sample is not maintained anymore. Check out its brand new version in CVCEditor SDK.

This sample shows how to configure CVCEditor instances to use the Placeholder plugin that lets you insert read-only elements into your content. To enter and modify read-only text, use the Create Placeholder   button and its matching dialog window.

To add a CVCEditor instance that uses the placeholder plugin and a related Create Placeholder   toolbar button, insert the following JavaScript call to your code: 

CVCEDITOR.replace( 'textarea_id', {
    extraPlugins: 'placeholder',
    toolbar: [ [ 'Source', 'Bold' ], ['CreatePlaceholder'] ]
});

Note that textarea_id in the code above is the id attribute of the <textarea> element to be replaced with CVCEditor.
